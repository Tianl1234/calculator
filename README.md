🧠 1. KI-Denkstruktur (Neural Link)
Die App nutzt eine Inferenz-Kette, um natürliche Sprache in Finanzdaten zu übersetzen. Sie erkennt automatisch Parameter wie Startkapital, Zinssatz und Laufzeit, ohne dass der Nutzer starre Formeln eingeben muss.

🛡️ 2. Sichere Mathematik-Engine (AST)
Um mathematische Sicherheit zu garantieren, wird jede Eingabe durch einen Abstract Syntax Tree (AST) validiert. Dies verhindert Code-Injection und sorgt für hochpräzise Berechnungen, indem mathematische Ausdrücke in eine Baumstruktur zerlegt werden.

💾 3. Lokales Gedächtnis (SQLite Persistence)
Das System verfügt über ein "ewiges Gedächtnis". Alle Nutzerpräferenzen und Sparziele werden in einer lokalen SQLite-Datenbank gespeichert.

Privacy First: Keine Cloud-Anbindung, alle Daten bleiben auf dem PC des Nutzers.

Proaktive Begrüßung: Die KI erkennt den Nutzer beim Start wieder und schlägt am letzten Punkt der Finanzplanung vor fortzufahren.
